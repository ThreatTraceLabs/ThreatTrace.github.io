Subfinder

Open Source

Fast subdomain discovery tool that uses passive sources to enumerate valid subdomains. Essential for mapping an organization's external attack surface.

Use Case:

Discover all subdomains of a target domain to identify forgotten services, staging environments, or misconfigured applications that might be vulnerable.

subfinder -d example.com -o subdomains.txt

Amass

OWASP

In-depth DNS enumeration and network mapping tool. Performs OSINT gathering, subdomain discovery, and DNS reconnaissance using multiple data sources.

Use Case:

Build comprehensive network maps of target organizations, discover hidden infrastructure, and identify relationships between domains and IP addresses.

amass enum -d example.com -o amass_output.txt

Aquatone

Open Source

Visual inspection tool for websites across large numbers of hosts. Takes screenshots of websites and provides a visual overview of HTTP-based attack surfaces.

Use Case:

Quickly review hundreds of discovered domains/subdomains visually, identify interesting targets, login panels, admin interfaces, or forgotten applications.

cat subdomains.txt | aquatone -out ./aquatone_results

SQLMap

Open Source

Automated SQL injection and database takeover tool. Detects and exploits SQL injection flaws across various database management systems.

Use Case:

Test web applications for SQL injection vulnerabilities, extract database contents, access file systems, and execute commands on the operating system.

sqlmap -u "http://target.com/page?id=1" --dbs --batch

ffuf

Open Source

Fast web fuzzer written in Go. Extremely versatile for directory brute-forcing, parameter discovery, virtual host enumeration, and more.

Use Case:

Discover hidden directories and files, enumerate API endpoints, find backup files, test for parameter pollution, and identify virtual hosts.

ffuf -w wordlist.txt -u https://target.com/FUZZ -mc 200

Nmap

Open Source

The definitive network scanning tool for security auditing and network discovery. Maps networks, identifies services, detects operating systems, and finds vulnerabilities.

Use Case:

Scan networks for open ports, identify running services and versions, detect operating systems, find potential vulnerabilities, and map network topology.

nmap -sV -sC -O -oA scan_results target.com

RustScan

Open Source

Modern, ultra-fast port scanner that feeds results directly into Nmap. Scans all ports in seconds while maintaining accuracy.

Use Case:

Quickly scan large networks or multiple targets for open ports, then automatically run detailed Nmap scans on discovered services for comprehensive enumeration.
